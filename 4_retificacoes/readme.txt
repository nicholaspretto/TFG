Para a base de massas d'água e de ruas em São Paulo, faça download
da opção Sudeste dos arquivos brasileiros do Geofabrik.de:
http://download.geofabrik.de/south-america/brazil.html

A referência para a localização dos rios em 1842 encontra-se no seguinte
livro:

CORREA, Felipe. São Paulo: a graphic biography = São Paulo: uma 
biografia gráfica. São Paulo: Romano Guerra Editora, 2018.

Para ter acesso à imagem-base para georreferenciamento, é possível
acessar uma versão disponível no site do coletivo Somatic:

https://www.somatic-collaborative.com/research/sao-paulo-a-graphic-biography
