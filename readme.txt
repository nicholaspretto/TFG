Nesta pasta, estão (quase) todas as bases de dados utilizadas no trabalho
final de graduação "Visualização de Dados para Problemas Urbanos: As águas
de São Paulo e a emergência climática".

Para referenciar corretamente as fontes, é necessário verificar no
próprio trabalho as fontes originais. Para isso, um link para o arquivo
no banco de trabalhos da USP será disponibilizado assim que disponível.

Existem três motivos para bases de dados estarem ausentes aqui:
1) O arquivo é muito pesado;
2) O arquivo é disponibilizado por alguma entidade ou instituição que
se beneficia do download direto do seu site;
3) Existe alguma incerteza sobre a possibilidade de divulgação pública
e irrestrita da informação, por direito autoral.

Em qualquer um dos casos, os links de referência foram disponibilizados.

Caso queira utilizar alguma das bases em seu trabalho, além de referenciar
as fontes originais, é de bom tom citar o presente trabalho:

KAMINSKI, N. Visualização de Dados para Problemas Urbanos: as águas
de São Paulo e a emergência climática. Trabalho final de graduação
(Arquitetura e Urbanismo) — Universidade de São Paulo (USP). São Paulo, 2022.
